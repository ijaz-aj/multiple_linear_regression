# Multiple Linear Regression with scikit-learn

Explore multiple linear regression with scikit-learn in this repository.

## Contents

- `Multiple_linear_regression-Economics Dataset.ipynb`: Jupyter Notebook for multiple linear regression.
- `economic_index.csv`: Sample dataset.

## Getting Started

1. Clone the repo: `git clone https://github.com/ijaz-aj/Multiple_linear_regression-Economics Dataset.git`
2. Open and run `Multiple_linear_regression-Economics Dataset.ipynb`.

## Usage

Follow the notebook to implement and understand multiple linear regression using scikit-learn.

## Dataset

`economic_index.csv` contains a sample dataset for model training and testing.

## Dependencies

- scikit-learn, numpy, pandas, matplotlib, seaborn
- jupyter (for running the notebook)
